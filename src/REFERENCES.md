## API Coverage
- Connection management (connect with SSL/TLS support)
- Query execution (parameterized queries, single-row queries)
- Schema inspection (databases, tables, columns, indexes)
- Table DDL (create, drop, alter)
- Data manipulation (insert, update, delete, upsert)
- Migration tracking (run and list migrations)
- Transaction management (begin, commit, rollback with isolation levels)
- Server diagnostics (table info, database size, active connections)
- Data export (CSV and JSON)
